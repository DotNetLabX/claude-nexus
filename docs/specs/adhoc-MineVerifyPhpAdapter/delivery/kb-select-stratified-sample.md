## Edge Cases

- Candidate count ≤ requested count: BR-1 — whole pipeline short-circuits, candidates returned reindexed-unchanged.
- Randomness: cell iteration order (BR-9) and largest-remainder tiebreaks (BR-15/17) come from the injected `Random\Randomizer` — tests must seed it (e.g. Xoshiro256StarStar with a fixed seed) for determinism.
- Zone/device values containing the literal `|`: BR-6 — no escaping in cellKey; distinct strata can collide.
